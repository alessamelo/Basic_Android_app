# AppBasicaAndroidStudio

Proyecto sencillo (Kotlin + XML) para primera clase.

## Qué hace
- Campo para nombre
- Contador con Incrementar/Reset
- Toast con nombre+contador
- Navegación a segunda pantalla enviando datos con Intent
- Logs del ciclo de vida en Logcat

## Compilar
```bash
chmod +x gradlew
./gradlew assembleDebug
```
APK: `app/build/outputs/apk/debug/app-debug.apk`
